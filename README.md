# 🚀 Zovira

Zovira is a full-featured web application built using **Node.js, Express, MongoDB, and EJS**. It provides a secure and scalable backend with user authentication, session management, and dynamic content rendering.

---

## 📌 Features

* 🔐 User Authentication (JWT & Sessions)
* 🔑 Password Hashing using bcrypt
* 📦 MongoDB Database Integration (Mongoose)
* 📁 File Uploads with Multer
* 🍪 Cookie & Session Management
* ⚡ Dynamic Views with EJS
* 🛠 Environment Configuration using dotenv

---

## 🛠 Tech Stack

* **Backend**: Node.js, Express.js
* **Database**: MongoDB (Mongoose)
* **Templating Engine**: EJS
* **Authentication**: JWT + Express Session
* **Other Tools**: Multer, bcrypt, dotenv

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Kaushal-Dubey7/Zovira.git
cd Zovira
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Setup environment variables

Create a `.env` file in the root directory and add:

```env
PORT=3000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
SESSION_SECRET=your_session_secret
```

---

## ▶️ Run the Application

```bash
npm start
```

Or (if no start script):

```bash
node app.js
```
